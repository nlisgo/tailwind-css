Understanding Breakpoints

Mobile-First Philosophy

Tailwind uses a mobile-first breakpoint system. Base utilities apply to all screen sizes, then you use responsive prefixes to override for larger screens:

  • Default - Applies to all screens (0px+)
  • sm: - Small screens and up (640px+)
  • md: - Medium screens and up (768px+)
  • lg: - Large screens and up (1024px+)
  • xl: - Extra large screens and up (1280px+)
  • 2xl: - 2X large screens and up (1536px+)

How to Add Custom CSS:

  1. Open src/css/style.css
  2. Add your custom CSS after the @import "tailwindcss"; line
  3. Run npm run build or npm run dev
  4. Your custom classes are now available to use alongside Tailwind!
